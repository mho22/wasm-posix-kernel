# Phase 13e: Exec Implementation Plan

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**Goal:** Implement `exec()` so a process can replace its image with a new Wasm binary while preserving PID, open file descriptors (minus CLOEXEC), environment, CWD, and signal mask.

**Architecture:** When a kernel calls `host_exec(path)`, the worker posts an `ExecRequest` to the host. The host loads the new Wasm binary, sends it back to the worker. The worker serializes exec-safe state from the current kernel via `kernel_get_exec_state()`, creates a new `WasmPosixKernel` with the new binary, and initializes it from the exec state via `kernel_init_from_exec()`. POSIX semantics: caught signal handlers reset to Default, SIG_IGN preserved, FD_CLOEXEC fds closed, pending signals preserved, signal mask preserved.

**Tech Stack:** Rust (kernel), TypeScript (host glue), Wasm

### Task 1: Exec State Serialization (Rust)

Add `serialize_exec_state()` and `deserialize_exec_state()` alongside existing fork serialization. The exec format is identical to fork but with different magic and different data:
- Magic: `0x45584543` ("EXEC")
- ppid: preserves current ppid (fork writes parent's pid)
- Signal handlers: only SIG_IGN preserved (caught Handler signals reset to Default)
- Pending signals: preserved (fork clears to 0)
- Blocked mask: preserved
- FD table: FDs with FD_CLOEXEC are excluded
- OFD table: only OFDs still referenced by remaining FDs

**Step 2: Add serialize_exec_state to fork.rs**

```rust
const EXEC_MAGIC: u32 = 0x45584543; // "EXEC"

pub fn serialize_exec_state(proc: &Process, buf: &mut [u8]) -> Result<usize, Errno> {
    let mut w = Writer::new(buf);

    // Header
    w.write_u32(EXEC_MAGIC)?;
    w.write_u32(FORK_VERSION)?;
    let total_size_offset = w.pos;
    w.write_u32(0)?;

    // Scalars — ppid stays the same (exec doesn't change parent)
    w.write_u32(proc.ppid)?;
    w.write_u32(proc.uid)?;
    w.write_u32(proc.gid)?;
    w.write_u32(proc.euid)?;
    w.write_u32(proc.egid)?;
    w.write_u32(proc.pgid)?;
    w.write_u32(proc.sid)?;
    w.write_u32(proc.umask)?;

    // Signal state — preserve blocked mask and pending signals
    w.write_u64(proc.signals.blocked)?;

    // Only preserve SIG_IGN handlers. Caught handlers (Handler) reset to Default per POSIX.
    let handlers = proc.signals.handlers();
    let ignore_count = handlers.iter().enumerate().filter(|(i, h)| {
        *i > 0 && **h == SignalHandler::Ignore
    }).count() as u32;
    w.write_u32(ignore_count)?;
    for (i, h) in handlers.iter().enumerate() {
        if i > 0 && *h == SignalHandler::Ignore {
            w.write_u32(i as u32)?;
            w.write_u32(handler_to_u32(*h))?;
        }
    }

    // Write pending signals (exec preserves them, unlike fork)
    w.write_u64(proc.signals.pending)?;

    // FD table — exclude FDs with FD_CLOEXEC
    use wasm_posix_shared::fd_flags::FD_CLOEXEC;
    let fd_entries: Vec<(i32, &FdEntry)> = proc.fd_table.iter()
        .filter(|(_, entry)| entry.fd_flags & FD_CLOEXEC == 0)
        .collect();

    // Collect referenced OFD indices
    let mut referenced_ofds: alloc::collections::BTreeSet<usize> = alloc::collections::BTreeSet::new();
    for (_, entry) in &fd_entries {
        referenced_ofds.insert(entry.ofd_ref.0);
    }

    w.write_u32(proc.fd_table.max_fds() as u32)?;
    w.write_u32(fd_entries.len() as u32)?;
    for (fd_num, entry) in &fd_entries {
        w.write_u32(*fd_num as u32)?;
        w.write_u32(entry.ofd_ref.0 as u32)?;
        w.write_u32(entry.fd_flags)?;
    }

    // OFD table — only include referenced OFDs
    let ofd_entries: Vec<(usize, &OpenFileDesc)> = proc.ofd_table.iter()
        .filter(|(idx, _)| referenced_ofds.contains(idx))
        .collect();
    w.write_u32(ofd_entries.len() as u32)?;
    for (index, ofd) in &ofd_entries {
        w.write_u32(*index as u32)?;
        w.write_u32(file_type_to_u32(ofd.file_type))?;
        w.write_u32(ofd.status_flags)?;
        w.write_i64(ofd.host_handle)?;
        w.write_i64(ofd.offset)?;
        w.write_u32(ofd.ref_count)?;
    }

    // Environment, CWD, rlimits, terminal — same as fork
    w.write_u32(proc.environ.len() as u32)?;
    for var in &proc.environ {
        w.write_u32(var.len() as u32)?;
        w.write_bytes(var)?;
    }
    w.write_u32(proc.cwd.len() as u32)?;
    w.write_bytes(&proc.cwd)?;
    for pair in &proc.rlimits {
        w.write_u64(pair[0])?;
        w.write_u64(pair[1])?;
    }
    w.write_u32(proc.terminal.c_iflag)?;
    w.write_u32(proc.terminal.c_oflag)?;
    w.write_u32(proc.terminal.c_cflag)?;
    w.write_u32(proc.terminal.c_lflag)?;
    w.write_bytes(&proc.terminal.c_cc)?;
    w.write_u16(proc.terminal.winsize.ws_row)?;
    w.write_u16(proc.terminal.winsize.ws_col)?;
    w.write_u16(proc.terminal.winsize.ws_xpixel)?;
    w.write_u16(proc.terminal.winsize.ws_ypixel)?;

    let total = w.pos as u32;
    w.patch_u32(total_size_offset, total);
    Ok(w.pos)
}
```

```rust
pub fn deserialize_exec_state(buf: &[u8], pid: u32) -> Result<Process, Errno> {
    let mut r = Reader::new(buf);

    let magic = r.read_u32()?;
    if magic != EXEC_MAGIC { return Err(Errno::EINVAL); }
    let version = r.read_u32()?;
    if version != FORK_VERSION { return Err(Errno::EINVAL); }
    let _total_size = r.read_u32()?;

    // Scalars
    let ppid = r.read_u32()?;
    let uid = r.read_u32()?;
    let gid = r.read_u32()?;
    let euid = r.read_u32()?;
    let egid = r.read_u32()?;
    let pgid = r.read_u32()?;
    let sid = r.read_u32()?;
    let umask = r.read_u32()?;

    // Signal state
    let blocked = r.read_u64()?;
    let handler_count = r.read_u32()?;
    let mut handlers = [SignalHandler::Default; 64];
    for _ in 0..handler_count {
        let signum = r.read_u32()?;
        let handler_val = r.read_u32()?;
        if (signum as usize) < 64 {
            handlers[signum as usize] = u32_to_handler(handler_val);
        }
    }
    let pending = r.read_u64()?;
    let signals = SignalState::from_parts_with_pending(handlers, blocked, pending);

    // FD table, OFD table, env, cwd, rlimits, terminal — same as fork
    // ... (identical parsing code)
}
```

```rust
pub extern "C" fn kernel_get_exec_state(buf_ptr: *mut u8, buf_len: u32) -> i32 {
    let proc = unsafe { get_process() };
    let buf = unsafe { core::slice::from_raw_parts_mut(buf_ptr, buf_len as usize) };
    match fork::serialize_exec_state(proc, buf) {
        Ok(written) => written as i32,
        Err(e) => -(e as i32),
    }
}
```

```rust
pub extern "C" fn kernel_init_from_exec(buf_ptr: *const u8, buf_len: u32, pid: u32) -> i32 {
    let buf = unsafe { core::slice::from_raw_parts(buf_ptr, buf_len as usize) };
    match fork::deserialize_exec_state(buf, pid) {
        Ok(proc) => {
            unsafe { *PROCESS.0.get() = Some(proc); }
            0
        }
        Err(e) => -(e as i32),
    }
}
```

```typescript
case "exec_reply": {
  const msg = m as ExecReplyMessage;
  // 1. Get exec state from current kernel
  const memory = kernel.getMemory()!;
  const EXEC_BUF_PAGES = 16;
  const startPage = memory.grow(EXEC_BUF_PAGES);
  const bufPtr = startPage * 65536;
  const bufSize = EXEC_BUF_PAGES * 65536;
  const getExecState = instance.exports.kernel_get_exec_state as (ptr: number, len: number) => number;
  const written = getExecState(bufPtr, bufSize);
  if (written < 0) {
    port.postMessage({ type: "error", pid: initData.pid, message: `kernel_get_exec_state failed: ${written}` });
    break;
  }
  const execState = new Uint8Array(memory.buffer, bufPtr, written).slice();

  // 2. Create new kernel with new binary
  const newKernel = new WasmPosixKernel(initData.kernelConfig, new NodePlatformIO(), callbacks);
  await newKernel.init(msg.wasmBytes);
  const newInstance = newKernel.getInstance()!;
  const newMemory = newKernel.getMemory()!;

  // 3. Write exec state to new kernel's memory and init
  const pages = Math.ceil(execState.byteLength / 65536) + 1;
  const sp = newMemory.grow(pages);
  const bp = sp * 65536;
  new Uint8Array(newMemory.buffer, bp, execState.byteLength).set(execState);
  const initFromExec = newInstance.exports.kernel_init_from_exec as (ptr: number, len: number, pid: number) => number;
  const result = initFromExec(bp, execState.byteLength, initData.pid);
  if (result < 0) {
    port.postMessage({ type: "error", pid: initData.pid, message: `kernel_init_from_exec failed: ${result}` });
    break;
  }

  // 4. Replace references
  kernel = newKernel;
  instance = newInstance;

  // 5. Notify host
  port.postMessage({ type: "exec_complete", pid: initData.pid } satisfies WorkerToHostMessage);
  break;
}
```

Note: The message handler needs to be async (or use .then()) for `newKernel.init(msg.wasmBytes)` which is async. Wrap the handler in an async IIFE.

```typescript
async exec(pid: number, wasmBytes: ArrayBuffer): Promise<void> {
  const info = this.processes.get(pid);
  if (!info || info.state !== "running") {
    throw new Error(`Cannot exec process ${pid}: not running`);
  }
  info.worker.postMessage(
    { type: "exec_reply", wasmBytes } as HostToWorkerMessage,
    [wasmBytes.slice(0)]
  );
  // Wait for exec_complete
  return new Promise<void>((resolve, reject) => {
    const timeout = setTimeout(() => reject(new Error("Exec timed out")), 10_000);
    const handler = (msg: unknown) => {
      const m = msg as WorkerToHostMessage;
      if (m.type === "exec_complete" && m.pid === pid) {
        clearTimeout(timeout);
        info.worker.off("message", handler);
        resolve();
      }
    };
    info.worker.on("message", handler);
  });
}
```
