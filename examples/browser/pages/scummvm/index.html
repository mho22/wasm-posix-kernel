ScummVM on wasm-posix-kernel

unmodified ScummVM 2.8 running in your browser via the SDL2 fbposix video driver and the kernel's /dev/fb0 + /dev/input/mice.

Pick a game to start.
Controls
Click canvas
Engage pointer-lock for mouse cursor
Esc
Quit menu (engine-defined)
F5
Save / load (engine main menu)
Ctrl + Q
Quit ScummVM

Audio is silent until /dev/dsp ships from the concurrent audio task. Game saves persist for the tab session only (in-memory VFS).
