ScummVM on wasm-posix-kernel

unmodified ScummVM 2.8 running in your browser, playing the freeware Beneath a Steel Sky. Video via /dev/fb0, audio via /dev/dsp, mouse via /dev/input/mice, all bridged by SDL2.

Click Start to boot ScummVM.
Controls
Left-click
Walk to / use object or hotspot
Right-click
Examine / look at
Space
Pause
F5
Save / Load / Quit menu
Esc
Skip cutscene / dialogue
. (period)
Skip current line

Click the canvas first — the browser will request pointer-lock so mouse motion can be relayed to the engine. Press Esc twice quickly to release pointer-lock.

Saves persist while this tab is open. Reload = fresh start.
